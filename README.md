# AgentHQ

**AI Agent Orchestration Platform** - Your command center for running entire businesses with AI agents.

## Features

- Central Dashboard - Real-time overview of all AI agents
- Agent Management - Configure agents from Claude, OpenAI, or custom providers
- Org Chart - Visual hierarchy of your agent workforce
- Ticketing System - Task assignment and tracking
- Budget Management - Control spending per agent
- Conversational AI - Agent-to-agent messaging
- Dark/Light themes

## Installation

```bash
git clone https://github.com/unn-Known1/agenthq.git
cd agenthq/frontend
pnpm install
pnpm run dev
```

## Desktop Builds

The project includes Electron configuration for building desktop apps:

```bash
cd electron
npm install
npm run build
```

For more details, see the [full documentation](https://github.com/unn-Known1/agenthq#readme).
