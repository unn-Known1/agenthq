## [1.0.0] - 2024-12-01

### Added
- Initial release of AgentHQ
- Central dashboard with real-time agent monitoring
- Agent management with multiple providers (Claude, OpenAI, Custom)
- Organizational chart with role-based hierarchy
- Ticketing system for task assignment
- Budget management per agent
- Agent-to-agent conversations
- Agent reporting hierarchy
- Tool system for file operations
- Dark and light theme support
- Desktop application (Electron) for Windows, macOS, and Linux
- CI/CD pipeline with GitHub Actions
- Automated releases

### Features
- **Dashboard**: Real-time stats, activity feed, and system logs
- **Org Chart**: Visual hierarchy with drag-and-drop organization
- **Tickets**: Create, assign, and track tasks
- **Budgets**: Monitor agent spending with visual charts
- **Conversations**: Agent messaging and collaboration
- **Logs**: Filterable system logs with severity levels
- **History**: Historical records of tasks, agents, and budgets
- **Settings**: Theme toggle, company mission, notification preferences
